Frontend Directory Structure Documentation

Tài liệu này mô tả cấu trúc frontend của ứng dụng web bán acc game. Dự án sử dụng React, Material UI (MUI), React Hook Form, @tanstack/react-query, Axios và JWT để xác thực. Dưới đây là giải thích chi tiết về chức năng của từng thư mục cùng với ví dụ minh họa.

Tổng Quan Dự Án

Dự án là một ứng dụng web để bán các loại acc game, sử dụng cấu trúc mô-đun với các công nghệ chính sau:

  • React: Xây dựng giao diện người dùng
  • Material UI (MUI): Hỗ trợ phát triển giao diện nhanh
  • React Hook Form: Xác thực biểu mẫu
  • @tanstack/react-query & Axios: Gọi API
  • JWT: Xác thực người dùng
  • JSON Server: Giả lập API trong quá trình phát triển

@utils

Chứa các hàm hỗ trợ như xử lý API, định dạng dữ liệu.

Ví dụ:

import axios from 'axios';

export const api = axios.create({
  baseURL: '/api'
});

export const formatPrice = (price) => `$${price.toFixed(2)}`;

// Sử dụng

api.get('/accounts').then(res => console.log(res.data));
console.log(formatPrice(50)); // $50.00

Các Tệp Gốc

App.jsx & App.css

Thành phần chính của ứng dụng và tệp CSS tương ứng.

main.jsx

Điểm khởi đầu của ứng dụng React.

env.example

Tệp ví dụ cho cấu hình biến môi trường.

.gitignore

Xác định các tệp/thư mục bị bỏ qua trong Git.

eslint.config.js

Cấu hình ESLint để kiểm tra mã nguồn.

index.html

Tệp HTML chính của ứng dụng.

package.json & package-lock.json

Quản lý dependencies của dự án.

README.md

Tài liệu hướng dẫn cài đặt và sử dụng dự án.

vite.config.js

Cấu hình Vite - công cụ build dự án.

Cấu Hình Alias

Dự án sử dụng alias để đơn giản hóa việc import: 

alias: {
  "@components": resolve(__dirname, "src/components"),
  "@contexts": resolve(__dirname, "src/contexts"),
  "@assets": resolve(__dirname, "src/assets"),
  "@pages": resolve(__dirname, "src/pages"),
  "@routers": resolve(__dirname, "src/routers"),
  "@utils": resolve(__dirname, "src/utils")
}

Tài liệu này sẽ giúp team của bạn hiểu rõ cấu trúc dự án và đóng góp hiệu quả. Nếu có thắc mắc, hãy xem README.md hoặc liên hệ với người phụ trách dự án.
